## 配置SSH公钥
用下面的命令查看`~/.ssh`目录下有没有`fileName.pub`文件。  
```bash
$ cd ~/.ssh
$ ls
```
如果没有，则用下方命令生成：  
```bash
$ ssh-keygen
```
位置可以默认，密码可以留空。

## 配置用户名和邮箱
每次Git提交时都会引用我们的用户名和邮箱，说明是谁提交的更新。
```bash
$ git config --global user.name "Stephen"
$ git config --global user.email stephen@example.com
```

## 初次提交更新
```bash
$ git clone url
$ git init
$ git add fileName
$ git commit -m "message"
$ git push
```
